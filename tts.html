Application Overview

This is a Flask-based Text-to-Speech (TTS) Web Application with a complete database system, audio processing, and professional UI.

Core Features

Text-to-Speech Conversion

  • Real-time Speech Synthesis: Convert text to speech instantly
  • Multiple Voice Support: Access all system voices available via pyttsx3
  • Adjustable Speech Speed: Speed control from 50 to 300 words per minute
  • Instant Playback: Speak text immediately through browser

Audio File Management

  • MP3 Export: Save speech as downloadable MP3 files
  • Temporary File Handling: Automatic cleanup of generated audio files
  • File Metadata Tracking: Store file size and creation details

Database System

  • SQLite Database: Lightweight, file-based database perfect for PythonAnywhere
  • Conversion History: Track all TTS conversions with timestamps
  • Audio File Registry: Maintain metadata for all saved audio files
  • Persistent Storage: Data persists between application restarts

Web Interface

  • Responsive Design: Works on desktop and mobile devices
  • Real-time Updates: Dynamic status messages and history loading
  • Professional UI: Gradient backgrounds, modern buttons, and clean layout
